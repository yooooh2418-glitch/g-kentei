# 戦績トラッカー PWA（メガハッサム×ゲッコウガ）

## 2通りの使い方

### A. そのまま単一ファイルで使う（最も簡単）
`index.html` をブラウザで開くだけ。記録はその端末に保存されます。
※ファイルを直接開く場合、Service Worker（オフライン）は環境により無効。記録機能は動きます。

### B. 本格的にPWA化（ホーム画面アプリ・オフライン対応）
このフォルダ一式（index.html / manifest.json / sw.js / icon-192.png / icon-512.png）を
**https のサーバーに置く**と完全なPWAになります。一番簡単なのは GitHub Pages：

1. GitHubで新規リポジトリ作成（例：`poke-tracker`）
2. このフォルダの中身を全部アップロード
3. Settings → Pages → Branch を `main` / `root` にして保存
4. 発行された `https://<ユーザー名>.github.io/poke-tracker/` をスマホのブラウザで開く

### ホーム画面に追加
- iPhone(Safari)：共有 ⬆ →「ホーム画面に追加」
- Android(Chrome)：メニュー ⋮ →「アプリをインストール」

## 機能
- 3指標（展開成功率/剣舞着地率/炎被弾率）の自動集計・目標達成判定
- 全期間＋直近10戦の達成率
- 連勝/連敗ストリーク
- 相手構築タグ別の勝率
- 勝敗・相手構築・反省メモの記録、履歴一覧
- 試合前チェックリマインド（炎受け選出忘れ防止）
- JSONバックアップ書き出し/読み込み（機種変対応）
