# メガ2枠 択ナビ ＆ 戦績ラボ｜ポケモンチャンピオンズ

ルカリオ＋ガブリアスのデュアルメガ軸を中心とした、選出択ナビ・メガ判断・戦績記録の単一HTML PWAアプリ。

## 構成
- `index.html` … アプリ本体（単一HTML / 依存はSheetJSとGoogle Fontsのみ、いずれもCDN）

自軍6匹：ルカリオ・ガブリアス・ヒスイゾロアーク・ウォッシュロトム・**アーマーガア**・マスカーニャ

## 使い方（インストール）

### スマホでアプリ化（PWA）
httpsで公開したURLを開き、ホーム画面に追加すると、アイコンから単独アプリとして起動できます。一度開けばオフラインでも動作します。

- **iPhone / Safari**：共有ボタン → 「ホーム画面に追加」（初回に自動で案内が出ます）
- **Android / Chrome**：右下「📲 ホームに追加」ボタン、またはメニュー →「アプリをインストール」

### PC
ブラウザでURLを開くだけ。Chromeならアドレスバー右のインストールアイコンからアプリ化も可能。

## 公開方法（どれか1つ）

PWA（オフライン動作・ホーム追加）には **https** での配信が必要です。`file://` での直接オープンだとService Workerが動かず、通常のWebページとしてのみ動きます。

### GitHub Pages（おすすめ・無料）
1. リポジトリを作成し `index.html` をアップロード
2. Settings → Pages → Branch を `main` / `root` に設定
3. 数十秒後に発行される `https://<ユーザー名>.github.io/<リポジトリ名>/` を開く

### その他
- Netlify / Cloudflare Pages にフォルダをドラッグ＆ドロップ
- 社内NASやサーバのhttps配信領域に置く

## データについて
- 戦績・設定は端末の **localStorage** に保存されます（サーバ送信なし）
- ブラウザのデータ消去・別端末では引き継がれません
- Excelエクスポート（SheetJS）でバックアップ・チーム共有が可能

## 備考
- 相手候補プリセットにはドオー等も「相手が使ってくる想定」として残してあります（自軍からは外し、アーマーガアに置換済み）
- Service Worker キャッシュ版数：`champ-dualmega-v2`
